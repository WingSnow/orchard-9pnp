# Orchard

2018年最佳PNP桌游的web版本。(WIP)

## 游戏规则

![rulebook](./public/resource/rulebook.png)

## 使用

```shell
# 安装
pnpm install
# 启动
pnpm run dev
```

## 功能说明

### 控制按键

- （在游戏桌）按住`右键`并移动：移动游戏桌
- （选择牌后）`空格`：顺时针旋转90°
- （选择牌后）`Esc`：取消选牌
